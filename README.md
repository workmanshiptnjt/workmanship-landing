# Workmanship Landing Page

A modern, minimalist landing page for the Workmanship faith-based mobile app.

## Features

- **Modern Design**: Clean black and white aesthetic with subtle gradient animations
- **Responsive**: Works seamlessly on desktop, tablet, and mobile devices
- **Accessible**: Built with proper semantic HTML and ARIA labels
- **Smooth Animations**: Scroll-triggered reveal animations with reduced-motion support
- **Navigation**: Quick-jump navigation to feature sections

## Structure

- `index.html` - Main landing page
- `style.css` - Styles with animations and responsive design
- `icon.png` - App icon/logo
- `index_Version2.html` - Alternative version (archived)
- `style_Version2.css` - Alternative styles (archived)

## App Features Highlighted

1. **Bible** - Scripture reading and exploration
2. **Bible Trivia** - Knowledge testing and quizzes
3. **Learn** - Reading plans and lessons
4. **Profile** - Progress tracking and achievements

## Deployment

This is a static website. Simply host the files on any web server or use services like:
- GitHub Pages
- Netlify
- Vercel
- AWS S3 + CloudFront

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Graceful degradation for older browsers
- Respects user's reduced-motion preferences

## Development

No build process required. Open `index.html` in a browser to view locally.
